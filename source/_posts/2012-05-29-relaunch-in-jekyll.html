--- layout: post title: Relaunch in Jekyll categories: [meta] --- So much for one month later. I'm relaunching the site using the admirable Jekyll for a few reasons:
  • I don't want to deal with spam comments anymore - discussion is easily available via social medias or direct through e-mail
  • The only dynamic feature I'm using other than comments is the sidebar, which I don't need
  • WordPress is kind of a beast, and switching to any static system will give me less moving parts to break / not update / leave vulnerable
There are going to be a few odds and ends as the site settles into place, but everything so far looks good. If you see any missing files or folders, don't hesitate to let me know!
